# Photo Gallery App

A simple photo upload gallery with captions.

## Deployment Instructions

To deploy this static HTML app for users:

### Option 1: GitHub Pages (Free)
1. Create a new repository on GitHub.
2. Upload the `index.html` file to the repository via the web interface.
3. Go to Settings > Pages, select "Deploy from a branch", choose main branch, and save.
4. Your site will be available at `https://yourusername.github.io/repositoryname/`

### Option 2: Netlify (Free)
1. Go to netlify.com and sign up.
2. Drag and drop the `index.html` file to deploy.
3. Your site will be live instantly.

### Option 3: Vercel (Free)
1. Go to vercel.com and sign up.
2. Upload the `index.html` file.
3. Deploy.

### Option 4: Local Hosting
- Run `python -m http.server 8000` in the directory and access at `http://localhost:8000/`
- For production, upload to any web server.
